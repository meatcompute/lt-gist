LightTable-Gist
===============

Gist the current buffer

Installation
============

gem install gist
Plugin Manager :: Install Gist


Development
===========

git clone to "~/Library/Application Support/LightTable/plugins/" (Mac OSX)

Bindings
========

Add,

  "<keybinding>" [:lt.plugins.gist/submit-gist]

in user.keymap, under editor
